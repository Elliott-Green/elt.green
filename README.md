# elt.green

Built with [Hexo](https://github.com/hexojs/hexo/blob/master/README.md) and [Icarus](https://ppoffice.github.io/hexo-theme-icarus/).

Start the server

```$ hexo server```

Create a new post

```$ hexo new "Hello Hexo"```

Generate static files

```$ hexo generate```

Clean

```$ hexo clean```

Deployment

```$ hexo deploy```
